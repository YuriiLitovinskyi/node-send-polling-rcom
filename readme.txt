Утиліта для відправки команди Опитування на всі ппк, які знаходились на зв'язку в момент запуску утиліти.
Запускається на хості з РКОМом.
РКОМ та Монго повинні бути запущені.
Команда відправляється по черзі з періодом 60 секунд.
Програма може використовуватися після переустановки РКОМу з чистою базою, для того щоб актуалізувати поточні стани приладів.
